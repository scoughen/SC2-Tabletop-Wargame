1. Game Overview & Setup

Game Summary

StarCraft 2 Tabletop is a fast-paced tactical wargame for 2 players simulating real-time strategy battles. Players build economies, construct buildings, produce armies, and fight for control of the battlefield.

Game Length: 10 rounds maximum (45-60 minutes)

Board Size: 3' × 3' (36" × 36")

Model Scale: 15-28mm miniatures

What You Need

  • 3' × 3' playing surface
  • Miniatures (or proxies) for units and buildings
  • Six-sided dice (d6) - multiple per player
  • Measuring tape or rulers (marked in inches)
  • Resource tracking sheet
  • Colored dice pool (different color per player) for activations
  • Tokens/markers for resource nodes, expansions, wounds, shields, etc.

Victory Conditions

Victory conditions vary by scenario. Common conditions include:

  • Annihilation: Destroy enemy main base
  • Control: Hold majority of objectives
  • Economic: Reach resource threshold
  • Timed: Most Victory Points at Round 10

Setup

Step 1: Choose Scenario

Players agree on a scenario or roll randomly.

Step 2: Set Up Board

  • Place 3' × 3' playing surface
  • Mark 7 base locations:
    • 2 starting bases (opposite corners)
    • 5 neutral expansions (center, mid-field × 2, forward × 2)
  • Each base location includes:
    • 6 mineral nodes (arranged in arc, 6" from base center)
    • 1 gas vent (at end of mineral arc, 6" from base center)

Step 3: Deploy Starting Forces

Each player receives:

  • 1 Main Base Building (Command Center/Nexus/Hatchery)
    • Placed in deployment zone (12" × 12" corner triangle)
    • At least 6" from board edges
  • 6 Workers (SCV/Probe/Drone)
    • Placed adjacent to main base
  • Starting Resources: 200 Minerals, 0 Vespene Gas

Deployment Zones:

Player 1: Bottom-left corner (12" × 12" triangle)

Player 2: Top-right corner (12" × 12" triangle)

Distance between main bases: ~33" diagonal

2. Turn Structure

Each round consists of three phases executed in order:

PHASE 1: Economy Phase (Simultaneous)

Both players collect resources simultaneously.

PHASE 2: Action Phase (Alternating)

Players alternate activating Command GroupsGroups of up to 12 combat units or 3 workers. Each group = 1 die in activation bag. using Order Dice system.

PHASE 3: Production Phase (Simultaneous)

Both players produce units, advance construction, and research upgrades.

3. Economy Phase

Both players execute simultaneously (~30 seconds):

Step 1: Collect Income

  • Main Base Building: +10 Minerals (automatic)
  • Workers on Minerals: +5 Minerals per worker adjacent to mineral node
  • Workers on Gas: +5 Gas per worker adjacent to gas extractor

Restrictions:

  • Maximum 1 worker per mineral node
  • Maximum 3 workers per gas extractor
  • Workers must be within 6" of a main base structure to collect resources

Step 2: Record Resources

Update resource totals on tracking sheet.

Example: 1 Command Center (+10M) + 4 workers on minerals (+20M) + 2 workers on gas (+10G) = +30M, +10G this round

4. Action Phase

Order Dice System

  1. Count Command Groups: Each player counts their Command Groups
    • Combat units: Groups of up to 12 models (1 die per group)
    • Workers: Groups of up to 3 workers (1 die per group)
  2. Place Dice: Each player places 1 colored die per Command Group into shared bag/cup
  3. Draw & Activate: Draw 1 die → that player activates 1 Command Group → repeat until bag empty

Command Group Activation

When your die is drawn, choose ONE Command Group and give it an ORDER. Each Command Group can only be activated once per round.

ADVANCE (Move + Shoot)

  • Move up to full Movement distance
  • Shoot at -1 To HitThe target number needed on a d6 to score a hit. Lower is better (3+ better than 5+). penalty OR don't shoot

FIRE (Shoot + Short Move)

  • Move up to half Movement distance
  • Shoot at normal accuracy

ASSAULT (Move + Melee)

  • Move up to full Movement distance
  • Engage in melee combat (must end in base contact)

ABILITY (Special Action)

  • Use unit-specific ability (Siege Mode, Stim, Blink, etc.)
  • Move up to half Movement distance
  • Cannot shoot

HOLD (Defensive Stance)

  • No movement
  • +1 to Armor SaveTarget number needed on d6 to prevent damage. Lower is better (3+ better than 5+). Maximum save is 3+. until next activation
  • Can use OverwatchReaction fire when enemies move within line of sight. Shoots at -1 To Hit before enemy completes action.

GATHER (Workers Only)

  • Move up to full Movement distance
  • Must end adjacent to mineral nodes or gas extractors
  • Must be within 6" of main base structure
  • Generates resources next Economy Phase

BUILD (Workers Only)

  • Move up to full Movement distance
  • Start or continue building construction
  • Workers can split from group to BUILD individually

REPAIR (Terran SCVs Only)

  • Move up to full Movement distance
  • Must end adjacent to damaged mechanical unit/building (within 0.5")
  • Pay 5 Minerals
  • Heal 1 wound on target

Overwatch (Reaction Fire)

  • Units on HOLD order can declare Overwatch
  • Place "Overwatch" marker next to unit
  • When enemy moves within line of sight AND range:
    • Overwatch unit shoots at -1 To Hit
    • Resolve before enemy completes action
    • Remove Overwatch marker after firing
  • Limitation: Only one Overwatch shot per activation

5. Movement Rules

Basic Movement

  • Units move up to their Move stat (in inches)
  • Can move in any direction
  • Can change facing freely during movement
  • Do not have to use full movement

Terrain Restrictions

Ground Units

  • Cannot move through: Buildings, terrain obstacles, other units (friendly or enemy)
  • Elevation changes: Must use ramps (cannot climb cliffs)
  • Must go around: Buildings and impassable terrain

Flying Units

  • Ignore: Buildings, terrain, elevation, other units
  • Can move freely over all obstacles

Special Movement

Cliff Jumping (Reapers, Colossi)

  • Ignore elevation changes
  • Move over cliffs as if flat ground
  • Still cannot move through buildings

Blink (Stalkers with upgrade)

  • Once per activation, teleport up to 8"
  • Ignores terrain, buildings, units, cliffs
  • Cannot blink into impassable terrain

Burrowed Movement (Roaches with Tunneling Claws)

  • Move at half speed while burrowed
  • Cannot be targeted without detection

Table Edge

  • Units cannot move off table edge
  • Units pushed toward edge stop at edge

6. Combat System

Ranged Combat Sequence

  1. Declare target
  2. Check range & line of sight
  3. Roll To Hit
  4. Apply StrikeBonus dice against specific unit types (Light, Armored, etc.). Strike hits ignore armor saves. dice (if applicable)
  5. Defender rolls saves
  6. Apply damage

Melee Combat Sequence

  1. Models in base contact (within 0.5")
  2. Roll To Hit
  3. Apply Strike dice (if applicable)
  4. Defender rolls saves
  5. Apply damage

Rolling To Hit

  1. Count attacking models
  2. Roll attack dice pool: (Attacks stat × number of attacking models)
  3. Each die meeting or exceeding "To Hit" value = 1 hit
  4. Proceed to Strike check, then saves

Modifiers:

  • Shooting after ADVANCE: -1 To Hit
  • Overwatch fire: -1 To Hit
  • HOLD order: Defender gets +1 to Armor Save

Strike Dice (Unit Counters)

Units with Strike abilities gain bonus dice vs specific unit types:

Strike Types:

  • Strike: Light - Bonus vs LightLightly armored units like Marines, Zerglings, Zealots. Vulnerable to Strike: Light attacks. units
  • Strike: Armored - Bonus vs ArmoredHeavy armor units like Marauders, Roaches, Stalkers. Vulnerable to Strike: Armored attacks. units
  • Strike: Biological - Bonus vs BiologicalOrganic units that can be healed by Medivacs. Includes most infantry and Zerg units. units
  • Strike: Mechanical - Bonus vs MechanicalRobotic units that can be repaired (Terran). Immune to biological effects. units
  • Strike: Massive - Bonus vs MassiveLargest unit classification. Very heavy units like Ultralisks, Thors, Colossi. units

Rules:

  • Add +1 die per attacking model vs target type
  • Strike dice hit on same To Hit number
  • Strike hits ignore armor saves
Example: 4 Marauders (Strike: Armored) attack Roaches (Armored)
Base: 4 models × 1 attack = 4 dice
Strike: +4 dice (1 per model)
Total: 8 dice, Strike hits ignore saves

Armor Saves

  1. Defender rolls 1 die per hit
  2. Each die meeting or exceeding Armor Save = damage prevented
  3. Failed saves = damage applied to Wounds
  4. Strike hits cannot be saved (ignore armor)

Protoss Shields

Protoss units have two defense layers:

  • Shields: Absorb damage first
  • Wounds: Damaged after shields depleted
  • Armor Save applies to both Shields and Wounds
  • Shields regenerate 1 point per Production Phase (if unit took no damage that round)

Damage & Casualties

  • Each failed save = 1 damage
  • Damage reduces Wounds (Shields first for Protoss)
  • When Wounds reach 0, model removed
  • Multi-wound models: Track damage with dice/tokens

Splash Damage

Some units hit multiple targets:

Types:

  • Radius Splash: Place template, all units within take automatic hits
  • Line Splash: All units along line take automatic hits

Rules:

  1. Attack primary target normally
  2. If hits scored, place splash template
  3. All units within template take 1 automatic hit each (no To Hit roll)
  4. Defenders save normally
Examples:
  • Siege Tank (Sieged): 3" radius splash
  • Colossus: 6" line splash
  • Baneling (death): 2" radius splash
  • Hellion: 8" line splash

Line of Sight

  • Must have clear line of sight to target
  • Buildings and terrain block LOS
  • Units do not block LOS (abstract)
  • Measure from attacker to any point on target

Range

  • Measure from closest point of attacker's base to closest point of target's base
  • Melee: 0-0.5" (base contact)
  • Ranged: Varies by weapon (6", 12", 18", 26", etc.)
  • Pre-measuring allowed at any time

Ground vs Air Targeting

  • Ground Only: Can only attack ground units
  • Air Only: Can only attack flying units
  • Ground and Air: Can attack both
  • FlyingUnits that ignore terrain for movement. Can only be attacked by anti-air weapons. units can only be attacked by anti-air weapons

7. Production Phase

Both players execute simultaneously (~1-2 minutes):

Step 1: Advance Construction

  • Buildings under construction advance 1 round
  • Most buildings complete after 1 round
  • Main bases (Command Center/Nexus/Hatchery) take 2 rounds
  • Completed buildings become active

Step 2: Building Regeneration

  • Zerg buildings: Regenerate 1 wound automatically
  • Protoss buildings: Regenerate 1 shield automatically
  • Terran buildings: Repaired by SCVs (costs minerals, done during Action Phase)

Step 3: Produce Units

  • Spend resources to produce units
  • Must have appropriate building
  • Units available NEXT round (not immediate)
  • Units deploy adjacent to producing building
  • Respect production building capacity limits

Production Limits (per building per round):

  • Barracks/Gateway/Spawning Pool: 4 infantry max
  • Factory/Robotics/Roach Warren: 2 vehicles max
  • Starport/Stargate/Spire: 1 air unit max
  • Command Center/Nexus/Hatchery: 3 workers max

Step 4: Research Upgrades (Optional)

  • Spend resources on permanent upgrades
  • Upgrades take 1 round to complete
  • Apply to all relevant units immediately when complete

Step 5: Build New Structures

  • Pay resource cost to start construction
  • Worker must use BUILD order during Action Phase
  • Note construction time (usually 1 round)

8. Buildings & Construction

Building Stats

Buildings have:

  • Armor Save: Defense value
  • Wounds/HP: Damage capacity
  • Shields (Protoss only): Regenerating defense layer
  • Production Capacity: Units produced per round
  • Unlocks: Units/buildings available after construction

Attacking Buildings

  • Buildings targeted like units
  • Most buildings cannot fight back
  • Defensive buildings can attack (Bunker, Photon Cannon, Spine Crawler, Spore Crawler, Missile Turret)

Construction Time

  • Most buildings: 1 round
  • Main bases: 2 rounds
  • Upgrades: Instant or 1 round

Building Placement

Terran

  • Can build anywhere on board (most flexible)
  • SCV must stay adjacent during construction
  • Cannot gather or fight while building
  • Most buildings can lift off and relocate

Protoss

  • Must build within Pylon power radius (9" from any Pylon)
  • Probe starts construction then leaves
  • Building auto-completes after 1 round
  • Unpowered buildings cannot function

Zerg

  • Must build on creepPurple organic carpet spreading 6" from Hatcheries and Creep Tumors. Required for Zerg building placement. (6" radius from Hatchery/Creep Tumor)
  • Drone consumed when building starts (remove drone)
  • Building auto-completes after 1 round
  • Buildings regenerate 1 wound per Production Phase

Defensive Buildings

Automated Defense: Defensive structures attack during both players' turns

  • Overwatch always active with no -1 To Hit penalty
  • Attack any enemy in range automatically

Detection: Some defensive buildings reveal cloaked/burrowed units

  • Missile Turret (Terran): 12" detection radius
  • Spore Crawler (Zerg): 12" detection radius
  • Photon Cannon (Protoss): 12" detection radius

9. Special Rules & Keywords

Unit Type Classifications

Size/Armor Types

  • Light: Vulnerable to Strike: Light (Marines, Zerglings, Zealots, Mutalisks)
  • Armored: Vulnerable to Strike: Armored (Marauders, Roaches, Stalkers, Tanks)
  • Massive: Very heavy units (Ultralisks, Thors, Colossi, Carriers)

Composition Types

  • Biological: Can be healed by Medivacs, affected by biological abilities
  • Mechanical: Can be repaired (Terran), immune to biological effects
  • Psionic: Rare, specific to caster units

Note: Units can have multiple types (Marine = Light + Biological)

Universal Abilities

Flying

  • Ignores terrain, buildings, elevation, units for movement
  • Can only be attacked by anti-air weapons
  • Cannot end movement on impassable terrain

Burrow (Zerg Upgrade)

  • Use ABILITY order to burrow/unburrow (costs full activation)
  • Burrowed units cannot move or attack (unless Tunneling Claws)
  • Cannot be targeted without detection
  • Detectors reveal burrowed units

Detection

Units/buildings that reveal cloaked/burrowed enemies:

  • Raven, Missile Turret (Terran)
  • Overseer, Spore Crawler (Zerg)
  • Observer, Photon Cannon (Protoss)

Regeneration (Zerg)

  • Heal wounds during Production Phase
  • Only if unit took no damage that round

Shield Regeneration (Protoss)

  • Regenerate 1 shield per Production Phase
  • Only if unit took no damage that round
  • Buildings regenerate shields automatically

Worker Mechanics

Gathering

  • Must be adjacent to mineral node or gas extractor
  • Must be within 6" of main base structure
  • Generate +5 resources per Economy Phase

Building Construction

See faction-specific sections for construction rules per faction.

Worker Combat

  • Workers have basic combat stats (weak)
  • Can fight back when attacked
  • Killing workers cripples economy (harassment strategy)

Command Groups

  • Maximum 12 models per combat Command Group
  • Maximum 3 workers per worker group
  • Each Command Group = 1 die in activation bag
  • All models in group must perform same order
